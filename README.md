# Fitur
- swap bolak balik di TeaSwap
- custom jumlah transaksi

**git clone script**
```
git clone https://github.com/tudeiy/teaswap.git
cd teaswap
```
**Create Screen**
```
screen -S teaswap
```
**INSTALL AUTO**
```
chmod +x teaswap.sh
./teaswap.sh
```
**Isi Private key dan alamat token (token telegram boleh kosongin aja)**
```
nano .env
```
**mainkan**
```
node teaswap.js
```
# Fungsi tambahan 
**cara masuk screen**
```
screen -r teaswap
```
